Contoh-contoh Dasar pada Pemrograman Jaringan dengan Python

Saya telah menyusun berbagai macam contoh dasar pada pemrograman jaringan. Tentunya menggunakan Python ya :). Saya menyusun ini dari berbagai sumber. Ada yang dari blog-nya orang, ada yang dari Python Docs, ada yang iseng saya buat sendiri. Selain tentang pemrograman berbasis jaringan, terdapat beberapa teknik dasar pemrograman Python (yang banyak digunakan dalam konteks jaringan) seperti iterasi dan pembacaan file juga saya tuliskan di sini. Berikut ini daftarnya berdasarkan topik.

  1. Tipe data list dan iterasinya: list, iterasi list
  2. Tipe data tuple (biasa digunakan untuk menyimpan nilai alamat IP dan port): tuple
  3. Tipe data dictionary dan iterasinya: dictionary
  4. Iterasi dasar: loop
  5. Percabangan dasar: branch
  6. Klien-server sederhana yang mengirim pesan teks: client1, server1, server2
  7. Membaca dan menulis file teks: read, write
  8. Klien-server dengan teknik select: client-select, server-select
  9. Klien-server dengan protokol UDP: server-udp, client-udp1, client-udp2
  10. Thread: thread1, thread2, thread3, thread4, thread5
  11. Echo-server dengan thread: echoserver-thread
  12. Klien HTTP dengan modul socket: http-client-socket
  13. Klien HTTP dengan urllib dan BeautifulSoup: urllib-bsoup
  14. Klien HTTP dengan urllib dan HTMLParser: urllib-htmlparser
  15. Klien HTTP dengan httplib: httplib-example
  16. Serialization dengan pickle: standalone, serialization-client, serialization-server
  17. XML-RPC: boolean-client, boolean-server, datetime-client, datetime-server, multicall-client, multicall-server

Sekian kumpulan contoh dasar source code pada topik pemrograman jaringan di Python. Kita sambung lagi kapan-kapan
